Основной целью и задачей проекта является проведения EDA для выбора параметров модели предсказывающей результаты экзамена по математике, исходя из данных о школьниках.
Это данные как о самих школьниках, так и о их родителях и семье, а также об учебе (в школе и вне ее).
В данном проекте проведены следующие этапы работы:

- Предобработка: составление функций для различных типов данных
- Первичный отсмотр данных
- Первичный анализ данных в столбцах
- Корреляционный анализ данных
- Анализ номинативных переменных
- Составление выводов
